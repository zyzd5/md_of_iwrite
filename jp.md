## 文法
* `〜よう`＋`としている`
    * `动词意志形`＋`としている`
    * `马上就要什么什么了`

* `〜において`
    * `名詞`＋`において`
    * `在什么方面`
## 敬語
### 尊敬語(主语是别人, 把别人拔高)
#### 特殊尊敬語
|動詞|尊敬語|
|-|-|
|`行きます`、`来ます`、`います`|`いらっしゃいますか`|
|`食べます`、`飲みます`|`召し上がります`|
|`言います`|`おっしゃいます`|
|`知っています`|`ご存じです`|
|`見ます`|`ご覧になります`|
|`します`|`なさいます`|
|`くれます`|`くださいます`|
#### 通用尊敬語
* `お〜になります`
    * `使います` -> `お使いなりますか`
    * `帰りました` -> `お帰りになりました`
    * `辞めます` -> `お辞めになります`

* `〜(ら)れます`
    * 一类动词
        * `休みます` -> `休まれます` 
        * `会います` -> `会いわれます`
        * `動詞(ます形の前)[い -> あの段]に変える`　＋　`れます`
    
    * 二类动词
        * `食べます` -> `食べられます`　
    
    * 特殊动词 
        * `来ます` -> `来られます`
        * `します` -> `されます`
        * `電話します` -> `電話されます`
### 謙譲語(主语是自己, 把自己降低)
#### 特殊謙譲語
|動詞|謙譲語|
|-|-|
|`行きます`、`聞きます`|`うかがいます`|
|`食べます`、`飲みます`、`もらいます`|`いただきます`|
|`います`|`おります`|
|`言います`(自己紹介)|`申します`|
|`します`|`いたします`|
|`行きます`、`来ます`|`参ります`|
#### 通用謙譲語
* `お〜します`
    * `お`＋動詞(`ます形の前`)＋`します`
    * `送ります` -> `お送ります`
    * `持ちます` -> `お持ちします`
    * `知らせます` -> `お知らせます`
    * `借ります` -> `お借りします`
    * 如果单词有特殊謙譲語用法的情况下不可以用通用语法

* `ご〜します`
    * 三类动词限定
    * `ご`＋名詞＋`します`
    * `説明します` -> `ご説明します`
    * `紹介します` -> `ご紹介します`
    * `` -> ``
    * `` -> ``

### 丁寧語(双方同样高度)
* `(で)ございます`
    * `名詞`＋`でございます`
    * `資料です` -> `資料でございます`

* `名詞にございます`
    * `名詞`＋`に`＋`ございます`
    * `横にあります` -> `横にございます`

* 名词中有关`お`、`ご`
    * 以前日本就有的名词一般＋`お`
    * 从中国传来的名词＋`ご`
    * 日常生活中经常的事情经常＋`お`
        * `お名前は？`、`お元気ですか？`、`お疲れ様です`
    * 片假名不加`お`、`ご`
    * 说自己时不加`お`、`ご`
## Asada School
### Day.1
* `AあってのB`
    * 因为有A, 所以B才会成立
> * お客様あっての商売
>   * [`お客様がいなければ商売はできない`]
>   * 有客人才有商业
* `Aあってのことだ`
    * [Aのおかげだ]
> * この企画が成功したのは皆さんの協力あってのことだ
>   * [`皆さんの協力のおかげで成功できた`]
>   * 这个计划能成功是多亏了大家

### Day.2
* `Aいかんだ`
    * `Aにかかっている`
    * 和A相关
> * 試験に合格できるかどうかは君の努力いかんだ
>   * [`努力する` -> `合格`、`努力しない` -> `不合格`]
>   * 考试是否合格与你的努力相关

* `Aいかんによらず`
* `Aいかんにかかわらず`
    * `Aの結果に関係なく、Aしてもしなくても`
> * 投票の結果いかんに関わらず、税金が引き上げられる
> * 和投票结果无关, 税金都会上涨
> * 
### Day.5

    * 表示xx的心情过于强烈

>   * 启动后就不可能停下来了

### Day.10
* `Aが早いかBした`
    * 做完A后立刻就做了B
> * 彼は会議が終わるが早いか、帰ってしまった
>   * 他开完会就直接回去了

### Day.14
* `〜ごとく`
    * ==`〜のように`、`〜みたいに`
> * 例のごとく彼女は遅れてやってきた
>   * `いつものように`[普段からよく遅れる]
>   * 她和往常一样迟到了

* `かのごとく`
    * 就好像什么一样
> * 赤信号に気づなっかた`かのごとく`
>   * 装作没看到地过了红灯

* `〜のように`/`〜みたいに` = `〜ごとく`＋形容词
* `〜のような`/`〜みたいな` = `〜ごとき`＋名词
* `〜のようだ`/`〜みたいだ` = `〜ごとし`＋。

> * `夢のように` = `夢のごとく時間が過ぎる`
> * `夢のよう` = `夢のごとき時間だった`
> * `夢のよう` = `夢のごとし`
### Day.18
* `〜ずくめ`
    * 状态完全为一种情况
        * 当用于表示颜色时只能用于表示黑色
    * 相近的语法: `〜だらけ`, `〜まみれ`
> * 現場では黒ずくめの男が目撃さてた
>   * 全身都是黑色的男人在现场被目击到了 
>
> * 最近の彼女はいい事ずくめだ
>   * 最近她每天都过得很好
>
> * この社会は規則ずくめで大変だ
>   * 这个社会只看规则就糟糕了

* `ずくめ`, `だらけ`, `まみれ` 的区别
    * `ずくめ` 只有在表示事物为一种状态时使用
        * `黒ずくめ`、`いいことずくめ`
    * `だらけ` 表示在空间内有很多这种东西
        * `傷だらけ`、`間違いだらけ`、`男だらけ`
    * `まみれ` 汚れ很多时使用
        * `血まみれ`、`泥まみれ`
